# deshaunh-creator.github.io
 🛡️ Cybersecurity Portfolio Website

* About
This is my personal cybersecurity portfolio built with HTML, CSS, and JavaScript. It showcases my skills, certifications, and security projects in a terminal-style interface.

 * Features
- Terminal-style UI with typing animation  
- Matrix rain background  
- Animated skill bars  
- Expandable project logs  
- Contact form with email integration  
- Scroll progress indicator  
- Resume download (.txt)
